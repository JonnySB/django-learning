for loop tags:

Or, you can use a loop tag to more dynamically print various data structures, for instance a list to a list:

    {% for item in some_list %}
  1. {{item}}
    2. {% endfor %}

Or, a dictionary to a list:

    {% for key, value in some_dict.items%}
  • {{key}}: {{value}}
    • {% endfor %}
